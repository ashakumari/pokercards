This program illustrates the poker card game by distributing a set of cards to a bunch of people and then identifying who really won the game.

Usage:
> py main.py
-- This will distribute 5 cards each to 3 players and then do a show on which set of card won the game
-- You can change the parameters of the deal function in the main.py file to change the number of players

> py test.py
-- This is to test all the various methods with varied set of data sets
